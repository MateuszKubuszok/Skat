# Skat game engine

Engine intented to be able to completely simulate Skat game, as well as act
in a bot-like manner.

## Installation

Download from https://bitbucket.org/MateuszKubuszok/skat.

## Usage

Run:

    $ java -jar skat-0.1.0-standalone.jar
